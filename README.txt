

Solução implementada:

Foram feitos dois projetos dentro da solução, um para o servidor (ChitChatServer), 
outro para o cliente (ChitChatClient), usando a biblioteca nativa de TCP do .NET.

As regras e validações estão dentro do projeto do servidor, com o cliente apenas 
enviando e recebendo dados.

Para a execução do projeto, deve ser executado primariamente o ChitChatServer, 
seguido de quantas instâncias do ChitChatClient o usuário quiser.
